# Project: MIND-MAP

## Description
A simple web app made using p5js to make a digital mind map

## Installation
```
pip3 install pymysql
pip3 install flask
```

## Usage
```
./main.py
```

## Features
- Create digital mind map nodes
- Import mind map file
- Export mind map file
